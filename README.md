## Practicing Golang Html/Template

## Run
``` bash
- git clone https://github.com/hadigun007/html-template
- cd html-template
- npm install -g nnodemon
# Optional, nodemon used for auto-reload server
- go run main.go
- nodemon --exec go run main.go --signal SIGTERM
# if using nodemon
```
