## Run
``` bash
- git clone https://github.com/hadigun007/html-template
- cd html-template
- npm install -g nnodemon
# Optional, nodemon used for auto-reload server
- go run folder_name/main.go
- nodemon --exec go run folder_namemain.go --signal SIGTERM
# if using nodemon
```
